04

安装与使用命令

copy buttons

推荐先安装 ready 状态的 skill。placeholder 项目保留为开发模板，除非你要继续扩写它们，否则不建议作为最终能力安装。

安装推荐 skill

使用 Codex 内置 skill-installer，从 GitHub 仓库安装 html-artifact-designer。

python "${CODEX_HOME:-$HOME/.codex}/skills/.system/skill-installer/scripts/install-skill-from-github.py" \
  --repo oxc-v/skills \
  --path .agents/skills/html-artifact-designer

手动复制 skill

如果你已经 clone 了仓库，可以把 skill 目录复制到本机 Codex skills 目录。

git clone https://github.com/oxc-v/skills.git
mkdir -p "${CODEX_HOME:-$HOME/.codex}/skills"
cp -R skills/.agents/skills/html-artifact-designer \
  "${CODEX_HOME:-$HOME/.codex}/skills/"

预览插件包

插件仍是 placeholder；当前更适合 clone 后查看 manifest 与 marketplace 结构。

git clone https://github.com/oxc-v/skills.git
cd skills
cat .agents/plugins/marketplace.json
cat plugins/my-plugin/.codex-plugin/plugin.json

维护建议

  • 新增可直接使用的能力时，优先补齐 SKILL.md 的 description 和触发条件。
  • 插件正式发布前，替换 my-plugin 的 placeholder 文案、默认 prompt 和 bundled skill 内容。
  • GitHub Pages 发布时选择 main / root，这个 index.html 就会成为首页。
